# Target User
Citizens from rural or semi-urban areas who want to report cybercrime incidents easily.

# User Flow Steps

1. User opens the CyberThana web portal.
2. User sees a simple landing page explaining the purpose of the portal.
3. User clicks on "Report Cybercrime".
4. User is redirected to the cybercrime reporting form.
5. User selects the type of cybercrime.
6. User enters incident details in simple language.
7. User selects the date of the incident.
8. User uploads supporting evidence (optional).
9. User submits the report.
10. System acknowledges the submission and generates a report reference ID (future scope).

# Key Design Considerations
- Minimal steps to reduce user confusion
- Simple language for rural accessibility
- Mobile-friendly design
